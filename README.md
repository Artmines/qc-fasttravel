# QC-Fast Travel
New Fasttravel for RSG Core to use with redemrp_menu_base for a more Red Dead 2 Look

# Dependancies
- rsg-core
- redemrp_menu_base

# Installation
- ensure that the dependancies are added and started
- add qc-fasttravel to your resources folder
